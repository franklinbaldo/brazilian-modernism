## 6. NOTES TO FUTURE-ME
Svelte 5 Context reactivity relies on passing getter functions (e.g., `setContext('key', () => value)`). During Astro's SSR build, these functions can sometimes be temporarily undefined or return undefined before hydration completes. Always provide a fallback (e.g., `getTabsName ? getTabsName() : 'fallback'`) in `$derived` runes that rely on context to prevent build failures.
