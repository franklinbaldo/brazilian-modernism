# Getting Started

Instale e use o COBOGÓ no seu projeto Astro+Svelte.

import DocsCallout from '../../components/docs/DocsCallout.astro';
import DocsCodeBlock from '../../components/docs/DocsCodeBlock.astro';

## Instalação

Instale via GitHub (até existir release no NPM):

<DocsCodeBlock lang="bash" filename="Terminal">
<pre><code>bun add github:franklinbaldo/cobogo#v0.1.0{"\n"}# ou: npm install github:franklinbaldo/cobogo#v0.1.0</code></pre>
</DocsCodeBlock>

<DocsCallout type="warning" title="Peer dependencies">
Cobogó precisa de `astro ^5 || ^6` e `svelte ^5`. Versões anteriores não suportam Runes.
</DocsCallout>

## Uso

### Importar tokens CSS

No layout principal:

```astro
---
import 'cobogo/styles';
---
```

Isso traz toda a paleta, tipografia, escala, radius e sombras como CSS variables.

### Importar componentes Svelte

<DocsCodeBlock lang="astro">
<pre><code>---{"\n"}import Button from 'cobogo/components/Button';{"\n"}import Card from 'cobogo/components/Card';{"\n"}import Badge from 'cobogo/components/Badge';{"\n"}---{"\n\n"}&lt;Button variant="azul"&gt;Ver manifesto&lt;/Button&gt;{"\n"}&lt;Card&gt;...&lt;/Card&gt;{"\n"}&lt;Badge&gt;Novo&lt;/Badge&gt;</code></pre>
</DocsCodeBlock>

### Componentes disponíveis

Componentes disponíveis incluem `Button`, `Card`, `Badge`, `ProfileCard`, `DataTable`, `SearchBar`, e os padrões cobogó.

<DocsCallout type="tip" title="Comece pequeno">
Importe 1 componente e veja se a tipografia herda corretamente antes de migrar páginas inteiras.
</DocsCallout>

### Configurar imports

<DocsCodeBlock lang="ts" filename="src/lib/cobogo.ts">
<pre><code>export &#123; default as Button &#125; from 'cobogo/components/Button';{"\n"}export &#123; default as Card &#125; from 'cobogo/components/Card';</code></pre>
</DocsCodeBlock>
