O `Alert` é utilizado para comunicar informações cruciais, avisos ou resultados de uma operação ao utilizador. Ele reflete de forma direta os tokens de semântica do sistema, permitindo leitura rápida sem distrair a jornada principal.

</div>

## Doutrina Curva & Concreto

Em COBOGÓ, os Alertas abraçam a "Curva" (bordas arredondadas) como elemento destacável de interface (como eles podem ser dispensáveis) enquanto mantêm a barra lateral de status como "Concreto" rígido, que dita a intenção.

Ele é derivado da mesma fundação visual que as Notificações (`Notification`), porém voltado a fluxos estáticos em linha, em vez de sobreposições flutuantes (toasts).

## Uso

- Utilize `variant="info"` (Azul) para informações normais.
- Utilize `variant="success"` (Verde) para operações bem-sucedidas.
- Utilize `variant="warning"` (Ocre) para alertas que exigem cuidado do usuário.
- Utilize `variant="danger"` (Vermelho) para falhas, erros ou ações destrutivas irreversíveis.

```svelte
<Alert variant="success" title="Documento Salvo" dismissible>
  As alterações no manifesto foram salvas com sucesso no servidor local.
</Alert>
```
