# MultiSelect

The `MultiSelect` component is a complex form organism that allows users to select multiple options from a dropdown list. It supports searching/filtering and integrates seamlessly with `FormField` for validation states.

## Basic Usage

The component requires an array of `options`, each containing a `value` and a `label`. It binds the selected values to an array of strings.

```svelte
<script>
  import MultiSelect from 'cobogo/components/MultiSelect';

  let selected = $state(['sp']);

  const options = [
    { value: 'sp', label: 'São Paulo' },
    { value: 'rj', label: 'Rio de Janeiro' },
    { value: 'mg', label: 'Minas Gerais' },
    { value: 'ba', label: 'Bahia' },
  ];
</script>

<MultiSelect
  bind:value={selected}
  {options}
/>
```

## Searchable

By setting the `searchable` prop to `true`, the `MultiSelect` component allows users to filter the options by typing into the input field.

```svelte
<MultiSelect
  bind:value={selected}
  {options}
  searchable
  placeholder="Search states..."
/>
```

## Validation States

Like other form controls in the COBOGÓ design system, `MultiSelect` supports `valid` and `invalid` states, adapting its styling to provide clear visual feedback adhering to the Curva & Concreto aesthetic.

It integrates seamlessly with the `FormField` wrapper, automatically inheriting validation states via Svelte Context.

```svelte
<FormField label="States" htmlFor="states-field" error="Please select at least one state">
  <MultiSelect
    id="states-field"
    {options}
  />
</FormField>
```
