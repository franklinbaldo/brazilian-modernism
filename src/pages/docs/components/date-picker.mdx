# DatePicker

O `DatePicker` é um componente de formulário para seleção de datas. Seguindo a doutrina *Curva & Concreto*, o campo de input adere à geometria de interação (curvas, `--r-1`), enquanto o *popover* do calendário obedece à geometria estática de dados concretos (sem arredondamento de bordas, apresentando um contorno de bloco sólido).

## Basic Usage

```svelte
<script>
  import DatePicker from 'cobogo/components/DatePicker';
  let date = $state('');
</script>

<DatePicker bind:value={date} />
```

<div class="not-content" style="margin-top: 2rem; margin-bottom: 2rem; padding: 2rem; border: 1px solid var(--border); border-radius: var(--r-soft); background: var(--bg-raised);">
  <DatePicker client:load />
</div>

## Inside a FormField

Como todos os componentes atômicos de formulário no COBOGÓ, o `DatePicker` consome automaticamente o contexto do `FormField` para gerenciar estados de erro (`invalid`) de forma acessível e visualmente assertiva, modificando o fundo do componente.

```svelte
<script>
  import DatePicker from 'cobogo/components/DatePicker';
  import FormField from 'cobogo/components/FormField';
  let date = $state('');
</script>

<FormField label="Data de Nascimento" htmlFor="dob-input" error="Você deve selecionar uma data válida." required>
  {#snippet children(props)}
    <DatePicker id={props.id} aria-describedby={props['aria-describedby']} bind:value={date} />
  {/snippet}
</FormField>
```

    <DatePicker client:load id="demo-dob" aria-describedby="demo-dob-error" />
  </FormField>
</div>
